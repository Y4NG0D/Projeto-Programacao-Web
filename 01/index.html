    HTML é a linguagem de marcação utilizada para criar páginas web.
    A estrutura básica de uma página HTML inclui as tags `html`, `head` e `body`, além de elementos como `title` e `meta`.
    O W3C é o Consórcio World Wide Web, uma comunidade internacional que desenvolve padrões para a web, incluindo HTML, CSS e outras tecnologias.
    CSS é a linguagem de estilo utilizada para definir a apresentação e o layout de páginas web escritas em HTML.
    JavaScript é uma linguagem de programação utilizada para tornar as páginas web interativas e dinâmicas.
    Programação web refere-se ao desenvolvimento de aplicações, sites e conteúdo web utilizando linguagens como HTML, CSS e JavaScript, entre outras tecnologias.
